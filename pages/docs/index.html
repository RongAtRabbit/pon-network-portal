GPON 详解

概述

GPON（Gigabit-capable Passive Optical Network）是ITU-T G.984系列标准定义的千兆无源光网络技术，是目前最主流的PON技术之一。

技术参数

参数 数值 说明
下行速率 2.488 Gbps OLT到ONU
上行速率 1.244 Gbps ONU到OLT
分光比 1:64（标准）/ 1:128（扩展） 支持的最大分光比
传输距离 20km（标准）/ 60km（扩展） OLT到ONU最大距离

协议栈结构

GPON协议栈采用分层架构：

  1. 物理层：GTC（GPON Transmission Convergence）层，负责帧传输和同步
  2. TC层：GEM（GPON Encapsulation Method）封装，支持多种业务承载
  3. 上层：Ethernet、TDM、ATM等业务层

XG/XGS-PON

XG-PON是10G GPON技术，ITU-T G.987系列标准定义。相比GPON，带宽提升4倍。

XG-PON vs XGS-PON

特性 XG-PON XGS-PON
下行速率 9.953 Gbps 9.953 Gbps
上行速率 2.488 Gbps 9.953 Gbps
带宽类型 非对称 对称

与GPON共存

XGS-PON可以与GPON在同一ODN上共存，通过WDM方式使用不同波长：

  • GPON：下行1490nm / 上行1310nm
  • XGS-PON：下行1577nm / 上行1270nm

50G PON

50G PON是下一代PON技术，ITU-T G.9804系列标准定义，带宽相比XGS-PON提升5倍。

技术规格

  • 下行/上行速率：49.766 Gbps（对称）
  • 下行波长：1342nm
  • 上行波长：1286nm
  • 共存能力：通过三模Combo光模块实现与GPON/XGS-PON共存

应用场景

  • 工业PON（时间敏感网络）
  • 5G前传
  • 企业专线
  • 数据中心互联

DBA 动态带宽分配

DBA（Dynamic Bandwidth Allocation）是PON系统的核心技术，实现上行带宽的动态分配，提高带宽利用率。

T-CONT类型

类型 业务类型 特点
Type 1 固定带宽 语音、专线，固定分配
Type 2 保证带宽 视频、企业应用，保证最小带宽
Type 3 保证+最大 互联网接入，有保证和最大限制
Type 4 尽力而为 普通上网，无保证带宽
Type 5 混合模式 综合业务，组合多种类型

技术演进路线

PON技术经历了多代演进，每一代都在带宽、距离、分光比等方面有显著提升。

演进历程

  1. 第一代 (BPON/APON)：基于ATM的PON技术，已被淘汰
  2. 第二代 (EPON/GPON)：1G/2.5G速率，当前存量主流
  3. 第三代 (10G PON)：10G速率，当前部署主流
  4. 第四代 (50G PON)：50G速率，2024-2026规模商用
  5. 第五代 (Beyond 50G)：100G/200G，研究阶段
