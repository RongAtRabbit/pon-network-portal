## 五、常见客户问题应答

### Q1: 中兴价格比华为低，为什么要选华为？

**A**: 
1. **长期TCO**: 华为50G PON演进能力，10年内无需更换设备
2. **运维效率**: iMaster NCE可降低60%运维成本
3. **可靠性**: Type B/C/D完整保护，减少业务中断损失
4. **生态兼容**: 与运营商现网OSS/BSS集成更成熟

### Q2: 华三说他们的EPON交换能力更强？

**A**:
1. **技术演进**: EPON已逐步被GPON/XGS-PON取代，华三XGS-PON支持有限
2. **PON专业性**: 华为PON原生功能（光功率、保护、切片）更完整
3. **企业网差异**: 华三擅长企业网交换，但PON是运营商级技术，华为积累更深

### Q3: 烽火是国产化，为什么选华为？

**A**:
1. **国产化**: 华为也是完全国产化，核心芯片自研
2. **技术领先**: 华为在全球PON市场份额第一，技术更先进
3. **三层能力**: 烽火不支持IS-IS/BGP，大型网络受限
4. **智能化**: 华为AI运维能力远超烽火

### Q4: 诺基亚是国际品牌，技术是不是更成熟？

**A**:
1. **市场地位**: 华为全球PON市场份额第一，超过诺基亚
2. **本地化**: 华为更懂中国客户需求，定制化能力强
3. **服务响应**: 华为本地化服务团队，响应速度更快
4. **性价比**: 同规格产品价格更优

## 六、招投标关键检查点

### 6.1 技术标书关键项

| 检查项 | 华为优势 | 注意事项 |
|--------|---------|---------|
| **PON技术标准** | 支持ITU-T G.984/G.987/G.9807 | 确保客户要求XGS-PON |
| **保护机制** | 要求Type B/C/D全支持 | 竞争对手可能只支持Type B |
| **网管协议** | 要求Telemetry/NETCONF | 传统厂商可能只支持SNMP |
| **安全加密** | 要求AES-128/GEM加密 | 部分厂商不支持 |
| **开放接口** | 要求RESTful API/SDK | 部分厂商API能力弱 |

### 6.2 商务标书策略

| 策略 | 说明 |
|------|------|
| **TCO分析** | 强调10年TCO，而非初始采购价 |
| **运维成本** | 量化AI运维节省的人力成本 |
| **扩展成本** | 强调软件升级无需换线 |
| **服务价值** | 强调本地化服务响应速度 |
