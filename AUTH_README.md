# PON Portal 权限管理说明

## 管理员账号

### 默认账号
- **用户名**: `admin`
- **密码**: `admin123`

### 安全提示
⚠️ 建议首次登录后修改默认密码（需要手动编辑 `js/auth.js` 文件）

---

## 权限控制说明

### 访客模式（未登录）
- ✅ 浏览所有页面
- ✅ 查看文档和技术资料
- ✅ 下载公开文档
- ❌ 上传文档
- ❌ 编辑内容
- ❌ 添加新技术

### 管理员模式（已登录）
- ✅ 浏览所有页面
- ✅ 上传文档到分类目录
- ✅ 编辑现有内容
- ✅ 添加新的技术模块
- ✅ 管理已上传文档

---

## 使用指南

### 登录步骤
1. 点击页面上的 **"🔐 登录"** 按钮
2. 输入管理员账号密码
3. 登录成功后显示 **"👤 admin (管理员)"**
4. 上传/编辑功能自动启用

### 登出步骤
1. 点击用户状态栏的 **"退出"** 按钮
2. 页面刷新后返回访客模式

## 修改管理员密码

编辑文件: `pon-network-portal/js/auth.js`

```javascript
const AUTH_CONFIG = {
    adminUsername: 'admin',
    adminPassword: '你的新密码',  // 修改这里
    sessionTimeout: 24 * 60 * 60 * 1000
};
```

## 会话管理

- 登录状态保存24小时
- 刷新页面不会丢失登录状态
- 清除浏览器缓存会登出
- 可以主动点击"退出"按钮登出

---

## 故障排除

### 无法显示登录框
- 检查 `auth.js` 是否正确加载
- 查看浏览器控制台错误信息

### 登录失败
- 确认用户名密码正确
- 检查浏览器是否禁用localStorage

### 登录后权限未生效
- 刷新页面重试
- 清除浏览器缓存后重新登录
